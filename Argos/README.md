# ARGOS AI

ARGOS AI is a premium, AI-first front-end prototype for a smart warehouse platform inspired by enterprise logistics systems.

## Features

- Executive dashboard
- Digital Twin visualization
- Load registration workflow
- AI allocation panel with explainable decisions
- Audit history view
- Risk and compliance modules
- Responsive enterprise-style UI

## Run locally

```bash
npm install
npm run dev
```

Then open (https://argos-wilson-sons.lovable.app/)
